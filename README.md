# GUI-приложение для аппроксимации данных методом наименьших квадратов (МНК)

Учебный проект (курсовая работа), представляющий собой десктопное приложение для проведения математической аппроксимации экспериментальных данных и их визуализации.

## 🛠 Технологический стек
* **Язык программирования:** Java (JDK 25)
* **Графический интерфейс (GUI):** Java Swing / AWT
* **Визуализация (Графики):** JFreeChart 1.5.5, JCommon 1.0.24
* **Среда разработки:** NetBeans IDE

## 🚀 Функционал приложения
* Расчет коэффициентов аппроксимирующей функции методом наименьших квадратов (МНК).
* Интерактивный ввод исходных данных через табличный интерфейс (Swing `TableModel`).
* Построение графиков исходных точек и полученной кривой аппроксимации на одной координатной сетке.
* Сохранение результатов расчетов (координат исходных и аппроксимированных точек) в текстовый файл с разделением табуляцией (TSV) для последующего анализа.

## 📁 Структура исходного кода
* `Variant20.java` — точка входа в приложение, запуск главного окна.
* `MainForm.java` — проектирование и логика работы графического интерфейса (обработка событий, таблица данных, интеграция графиков).
* `MNK.java` — математический модуль, реализующий расчеты по методу наименьших квадратов.
* `FileSaver.java` — вспомогательный класс для экспорта расчетных данных в файл.

## ⚙️ Запуск проекта
Для сборки и запуска проекта вам потребуются установленный JDK и библиотеки `JFreeChart` и `JCommon` (находятся в папке `lib`).

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/DanilD12/least-squares-approximation-gui.git
2. Откройте проект в вашей IDE (NetBeans, IntelliJ IDEA и др.).
3. Подключите внешние библиотеки .jar из папки lib к проекту (в NetBeans: Libraries -> Add JAR/Folder).
4. Запустите главный класс coursework.Variant20.
